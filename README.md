# PyTetris
A fully playable implementation of Tetris, the classic block falling puzzle, written in Python with Pygame

## Installation
### macOS
```bash
git clone https://github.com/sujpac/pytetris.git
cd pytetris
sudo python3 -m pip install pygame
python3 pytetris.py
```
